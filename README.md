# 👕 MoodFit — AI OOTD Feedback Service

> 사진 한 장으로 오늘의 코디를 분석하고, 스타일 피드백과 개선 포인트를 제안하는 AI 패션 스타일링 서비스

---

## 📌 프로젝트 소개

MoodFit은 사용자가 업로드한 OOTD(Outfit Of The Day) 사진을 분석하여 스타일 분위기, 컬러 조합, 어울리는 장소, 개선 포인트 등을 피드백해주는 AI 기반 패션 서비스입니다.

기존 패션 추천 서비스는 사용자가 보유한 의류를 일일이 등록하거나 복잡한 정보를 입력해야 하는 경우가 많습니다.

MoodFit은 **"지금 입은 옷 사진 한 장"**만으로 쉽고 빠르게 스타일 피드백을 받을 수 있도록 설계되었습니다.

---

## 🎯 문제 정의

- 매일 입은 코디가 괜찮은지 객관적으로 판단하기 어렵다.
- 패션에 관심은 있지만 전문적인 피드백을 받을 기회가 적다.
- 기존 의류 관리 서비스는 초기 입력 비용이 높아 사용 진입장벽이 존재한다.
- 사진 한 장으로 간편하게 스타일 분석을 받고 싶은 수요가 존재한다.

---

## 👥 대상 사용자

- 자신의 코디가 어울리는지 궁금한 사람
- 데이트, 학교, 출근 등 상황별 스타일 피드백을 받고 싶은 사람
- 패션에 관심은 있지만 전문 지식은 부족한 사람
- SNS 업로드 전 객관적인 의견을 듣고 싶은 사람

---

## ✨ 핵심 기능

### 1. OOTD 사진 분석

- 사용자가 착용한 코디 사진 업로드
- AI가 스타일 분위기 및 특징 분석

### 2. 스타일 피드백 제공

- 코디의 장점 분석
- 개선 포인트 제안
- 자연스러운 스타일링 조언 제공

### 3. 스타일 키워드 추천

예시)

- #캐주얼
- #미니멀
- #데일리룩
- #데이트룩
